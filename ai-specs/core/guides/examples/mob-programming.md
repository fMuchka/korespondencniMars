# Mob Programming Workflow

**Real-time collaborative coding - you navigate, agent drives**

## Overview

Highest collaboration level where you and the agent work together in real-time:

- **You (Navigator)**: Guide the approach, make decisions
- **Agent (Driver)**: Write the code based on your guidance
- Continuous discussion and refinement
